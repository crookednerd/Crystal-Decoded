# 🎨 Crystal Decoded

**An interactive personal profile site built from my Insights Discovery® assessment results.**

Type 129: Creative Supporting Helper (Classic) — one of the ~9% of people who land in the rare "Creative" position on the Insights wheel.

## What Is This?

A mobile-first, single-page website that takes the 21-page PDF from my Insights Discovery profile and turns it into something I can actually *use*. It breaks down my colour energy mix, strengths, triggers, blind spots, and more into digestible, scrollable sections — with some data visualizations thrown in.

## Sections

| Section | What It Covers |
|---|---|
| **Energy Mix** | My four colour percentages with animated bars |
| **Creative Type** | What it means to be in the rare Creative position |
| **Strengths** | Key gifts I bring to teams and work |
| **Motivators** | What genuinely lights me up |
| **Triggers** | Stress signals and what sets them off |
| **Feedback** | How I receive criticism (and how to deliver it) |
| **Relationships** | How I show up in connections with others |
| **Blind Spots** | Patterns I might not see but others do |
| **Conscious Gap** | Differences between my conscious and less conscious personas |
| **Growth** | Development areas and stretch goals |
| **Cheat Sheet** | Do's and don'ts for working with me |
| **Management** | My natural leadership style and ideal environment |
| **Work Life** | How my profile plays out professionally |
| **Personal Life** | How it shows up off the clock |

## Tech

Just a single `index.html` file. No build tools, no frameworks, no dependencies to install.

- **Fonts:** Unbounded + Nunito (loaded from Google Fonts)
- **Design:** Dark theme, custom colour palette, grain texture, scroll-triggered animations
- **Layout:** Mobile-first, responsive up to desktop

## About Insights Discovery®

Insights Discovery is a psychometric tool based on Carl Jung's theory of psychological types. It uses four colour energies — Cool Blue, Earth Green, Sunshine Yellow, and Fiery Red — to map personality preferences and behavioural tendencies. Everyone carries all four energies in different proportions.
